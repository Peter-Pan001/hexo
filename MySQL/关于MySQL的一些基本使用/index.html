关于MySQL的一些基本使用

发表于 2020-10-25

首页 MySQL

MySQL数据库软件

关于配置（conf）

  • MySQL服务的启动与停止
    • 手动通过应用程序启动
    • 或者通过cmd配置命令启动
      • cmd -> services.msc ->MySQL
    • 或者cmd管理员模式启动
      • net stop MySQL
      • net start MySQL

关于登录与退出

  • 一般的MySQL版本：

    • 登录命令
      • cmd中，在mysql安装目录下输入：mysql -uroot -p密码
      • 然后根据提示输入密码（为此前安装时设置的密码）
    • 退出：
      • 直接输入exit或者quit即可

  • 如果是MySQL8.0版本的：

    • 在特定的SQL Commands line client 窗口中直接输入密码登录

关于目录结构

  • MySQL安装目录

  • bin：存放一些二进制文件以及一些exe文件，服务类和基础类的
  • data：存放日志文件、数据文件的
  • share：存放MySQL一些错误数据
